# NiftyLens

> **India’s sharpest stock market dashboard.** Real-time NSE/BSE prices, technical indicators, F&O option chain, and smart alerts in one place.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Features
- 📈 Live candlestick charts (1m to 1W)
- 📊 RSI, MACD, Bollinger Bands, EMA
- 🔔 Price alerts via Telegram and email
- 📂 F&O Option Chain viewer
- 💼 Portfolio P&L tracker
- 🔍 Stock screener

## Tech Stack
`Next.js` · `TypeScript` · `Python FastAPI` · `Supabase` · `TradingView Charts`
